【python-0002】从dos窗口运行python程序

发表于 2019-03-20 | 更新于 2019-03-22

今天教大家另外一个运行python程序的方法：从dos窗口运行python
点击屏幕左下角“开始”按钮

搜索框内搜索“cmd”，搜索结果里面，找到“cmd.exe”，打开它

输入”python”，再按下回车键，即进入python运行环境，看到>>>提示符

输入 print(1+1) ，再按下回车键，即可看到运行结果

小朋友，试着看看python能不能做更难的题目，比如
print(100+501)
print(102*145)
print(45/5)
print(45-6)
