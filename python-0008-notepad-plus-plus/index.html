【python】安装Notepad++

发表于 2019-03-25 | 更新于 2019-03-28

什么是Notepad++

Notepad++是 Windows操作系统下的一套文本编辑器
Notepad++功能比 Windows 中的 Notepad(记事本)强大
Notepad++是免费软件，可以免费使用
Notepad++的吉祥物是一只变色龙。

安装Notepad++

打开Notepad++的官方网址：https://notepad-plus-plus.org/

点击download按钮，进入下载界面

点击Download 64-bit x64下面的Notepad++ Installer 64-bit x64

下载之后，双击安装程序，开始安装，选择中文（简体）

下一步

选择“我接受”

点击“下一步”

点击“下一步”

勾选“Create Shortcut on Desktop”,这样会在桌面添加一个快捷方式。点击安装。

安装成功。点击完成

使用Notepad++编辑python文件

右键以前保存的python程序文件，选择Edit with Notepad++，这就进入了编辑界面

如果文件扩展名是.py
notepad++自动按照python的语法把你的代码高亮显示。
